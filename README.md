This project implements a line follower robot using ESP32 and embedded C/C++. The robot detects a path using IR sensors and adjusts motor speed for accurate real-time tracking.
Hardware Used

ESP32 microcontroller
IR sensor array
Motor driver (L298N)
DC motors and wheels
Battery and chassis

Features

Real-time line tracking
Forward, left, and right motion control
PWM-based motor speed control
Embedded C/C++ implementation

Working

The robot reads sensor values and dynamically adjusts motor speed to stay aligned with the path.
